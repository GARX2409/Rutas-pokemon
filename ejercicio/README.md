# Ejercicios de rutas

Este proyecto muestra ejercicios en JavaScript donde:
- Se usa `const` para un mapa de rutas.
- Se usan `let` en bucles para iteraciones.
- Se contrasta `var` y `let` en un `for` con `setTimeout`.
- Se genera un `map.html` con un SVG que dibuja las rutas visitadas en orden.

Instrucciones (PowerShell en Windows):

```powershell
# Ejecutar el script
node .\app.js

# Abrir el mapa generado en el navegador (comando Windows)
start .\map.html
```

Al ejecutar `node app.js` se mostrará el orden de visita en la consola y se generará `map.html` en la misma carpeta. Abre `map.html` para ver el mapa con las rutas y el orden de visita.
